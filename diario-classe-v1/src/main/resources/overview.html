Documentação do Projeto Diário de Classe

Bem-vindo à documentação do projeto Diário de Classe (diario-classe-v1).

Este projeto contém as seguintes funcionalidades principais:

  • Gerenciamento de alunos
  • Registro de avaliações
  • Manutenção de diários de classe
  • Gestão de encontros e frequências
  • Vinculação de alunos a diários

Os principais pacotes e suas responsabilidades são:

  • dominio: Contém as entidades principais do sistema.
  • dominio.base: Contém a classe base abstrata para as entidades.
  • dominio.enums: Contém as enumerações utilizadas pelo sistema.
  • repositorio: Contém os repositórios para acesso aos dados.
